# TAXSIM-35 Executable

The TAXSIM validator requires the TAXSIM-35 executable from NBER.

## Download

Download from: https://taxsim.nber.org/taxsim35/

Choose the appropriate version for your OS:
- macOS: `taxsim35-osx.exe`
- Linux: `taxsim35-unix.exe`
- Windows: `taxsim35-windows.exe`

Place the executable in this directory.

## Supported Years

TAXSIM-35 supports tax years 1960-2023 only.

## Usage

The validator will automatically find the executable if placed here.
